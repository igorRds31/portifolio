Jogo do Numero Secreto


Descrição: Este é um jogo simples feito com HTML, CSS e JavaScript, no qual o usuário deve adivinhar um número secreto gerado aleatoriamente pelo sistema. A interface permite que o jogador insira palpites, receba feedback sobre se o palpite está muito alto ou muito baixo, e descubra se acertou.

Jogo do Amigo Secreto


Descrição: Aplicativo web simples que permite realizar o sorteio de amigo secreto. Desenvolvido com HTML, CSS e JavaScript puro, sem uso de bibliotecas externas, com foco em usabilidade básica e lógica funcional do sorteio.
